## 📝 使用建议

### 运行脚本
```bash
# 演示程序
python scripts/run_demo.py

# YOLO 摄像头
python scripts/run_yolo_cam.py

# 运行测试
bash scripts/run_tests.sh  # Linux/Mac
scripts/run_tests.bat       # Windows
```

### 查看文档
- 主文档：`README.md`
- 文档中心：`docs/README.md`
- 项目报告：`docs/reports/`
- 目录结构：`DIRECTORY_STRUCTURE.md`

### 开发工作流
1. 源代码：`src/rws_tracking/`
2. 测试代码：`tests/`
3. 配置文件：`config.yaml`
4. 模型文件：`models/`
